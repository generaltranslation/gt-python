## Getting Started

Python libraries for General Translation's internationalization (i18n) platform. These packages provide server-side i18n for Python web frameworks like Flask and FastAPI.

- Visit our [documentation](https://generaltranslation.com/docs) to get started.
- Create a free API key at [generaltranslation.com](https://generaltranslation.com).
